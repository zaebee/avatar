# asi:one Web Encryptor

Шифратор для отправки контента в Instagram через email.

## Использование

1. Откройте `index.html` в браузере
2. Введите ваш секретный ключ (shared secret)
3. Введите текст поста
4. Прикрепите до 2 фотографий (будут сжаты до 1MB)
5. Нажмите "Зашифровать"
6. Скопируйте зашифрованный текст
7. Отправьте письмо с зашифрованным текстом в теле

## Безопасность

- Используется AES-256-GCM
- PBKDF2 (100,000 итераций) для генерации ключа
- Уникальный IV для каждого шифрования
- Соль генерируется случайно
- Ключ никогда не передаётся

## Требования

- Современный браузер с поддержкой Web Crypto API
- JavaScript должен быть включён
